Setup

Configuración

Sin conectar
  1. Sube estos archivos a GitHub y activa GitHub Pages.
  2. Copia la Redirect URI de abajo y pégala en tu app de Spotify.
  3. Pega tu Spotify Client ID y conecta la cuenta.
  4. Crea tu key gratis en workers.cloudflare.com (solo email, sin tarjeta) y pégala en el campo de abajo.

Crea el Worker gratis en workers.cloudflare.com (solo email, sin tarjeta) y pega el código worker.js que viene en el repo. Luego copia aquí la URL que te da Cloudflare.

Cuando publiques este repo en GitHub Pages, aquí aparecerá la URL exacta para registrar en Spotify.
