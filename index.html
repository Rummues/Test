Setup

Déjalo listo para GitHub Pages

Sin conectar
  1. Sube estos archivos a un repo nuevo en GitHub.
  2. Activa GitHub Pages desde la rama principal.
  3. Pega la URL publicada en el dashboard de Spotify como Redirect URI.
  4. Vuelve aquí, pega tu Client ID y conecta la cuenta.

Cuando abras este sitio desde tu URL de GitHub Pages, esa misma dirección será la Redirect URI que debes registrar en Spotify.
